# Snake
大家熟悉的經典小遊戲-貪吃蛇<br/>
JS練習Project<br/>
利用 canvas標籤, 搭配JS編寫而成<br/>
內涵貪吃蛇遊戲各式特色:<br/>
1.毎吃到果實身體會變長一格<br/>
2.果實被吃掉後,會在遊戲框範圍內隨機生成新果實<br/>
3.蛇可穿牆而過,從映射位置鑽出來<br/>
4.蛇頭碰觸到身體部分,即陣亡 GAME OVER<br/>
5.計分系統,吃一果實得一分與統計所有玩家最高分數<br/>
